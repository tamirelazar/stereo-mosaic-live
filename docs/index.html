What this is

A panorama whose viewpoint you set after the shot.

One sideways pan in. Every output column is sampled from a single source frame; the sampling mode decides which column per frame, so projection and viewpoint stay adjustable after capture. The dial shifts the base column.

pushbroom
Fixed column — one slit, flat strips. The original repo.
xslit
Column slides with frame index — perspective falls out. This fork.
forward
Column sweeps quadratically — a looming push-in.

X-slit is incidentally stereo: depth survives in the slope of each feature's trail across the frames. Reading it back is the next step — not built yet.
