Same frames, two slits

Pushbroom is the original repo: one fixed slit, flat strips. X-slit is this fork — the slit slides across the frames and perspective falls out of it. Same footage, one toggle apart.

How it works

The whole thing is a choice of columns.

Each output column is taken from one source frame. The mode picks which source column to sample per frame: pushbroom holds a fixed column (a single slit); xslit slides the column linearly with frame index (a virtual second slit → perspective); forward sweeps it quadratically (a looming push-in). The viewpoint dial shifts the base column. Tick "show slices" above to see the per-frame strip boundaries.

Depth — next

X-slit is quietly stereo — the sweep that bends perspective also leaves depth in the slope of each feature's trail across the frames. Reading it back is the next step. Not built yet.
